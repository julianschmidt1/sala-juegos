## Tecnologías utilizadas
- Angular
- SCSS
- PrimeNG

## Descripción del proyecto
Aplicación desarrollada con Angular utilizando componentes standalone, rutas y estilos personalizados con SCSS. Se implementó una interfaz moderna utilizando PrimeNG.
